让 Python 打包成 EXE，像点一个按钮一样简单

PyExe Studio 是一个面向 Windows 的可视化打包工具。你只需要选择 `.py` 文件和输出目录，就能调用 PyInstaller 完成构建，并实时查看日志与错误建议。

核心功能

一键打包

支持 `--onefile` 和 `--onedir`，覆盖常见发布方式。

运行模式切换

可切换 `--windowed` / `--console`，适配 GUI 与命令行项目。

UPX 压缩

支持手动设置 UPX 路径，也可自动使用内置 `UPX.EXE`。

实时日志

构建过程逐行输出，常见错误会给出可读的修复提示。

双语界面

中文 / English 切换，团队沟通和交付更直观。

浅色/深色主题

支持跟随系统主题，也可手动指定视觉风格。

3 步完成打包

  1. 选择 Python 源文件（`.py`）和输出目录
  2. 设置参数（名称、图标、模式、UPX、管理员权限）
  3. 点击开始并在日志区域观察构建结果

支持的常用参数

--onefile --onedir --windowed --console --name --icon --uac-admin --clean -y --upx-dir

常见问题

提示找不到 PyInstaller 怎么办？

先执行 `pip install pyinstaller`，然后重启软件再试。

出现 ModuleNotFoundError 怎么处理？

根据报错安装缺失依赖，例如 `pip install `。

权限不足或 Access is denied？

可尝试启用管理员权限选项，或以管理员方式运行软件。
